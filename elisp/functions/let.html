let

  • assigns variables in local scope
  • first takes list of variable value pairs, which should each be lists
  • then takes function list of what to do with variables
  • e.g. (let ((me "James")(wife "Indira")(daughter1 "Amara")(daughter2 "Isabelle")) (message "My name is %s, my wife is %s and my daughters are %s and %s." me wife daughter1 daughter2))
  • if you simply include variable names they will be set to nil
  • the tutorial recommends using ;;; -*- lexical-binding: t -*-
